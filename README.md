# Claude-Skills-for-Business
A collection of skills created by us, alongside hand-picked community skills we've reviewed and selected for quality.

---

## What are skills?

Skills are reusable instruction sets that extend what Claude can do. Each skill gives Claude specific knowledge, context, or behaviour for a defined task — from generating documents to applying brand guidelines.

---

## What's in this repository?

| Type | Description |
|---|---|
| **HybrIT skills** | Built and maintained by the HybrIT NZ team |
| **Community skills** | Sourced from the Claude community, reviewed and selected by us |

Community skills are reviewed for quality and relevance before being added. Some are included as-is; others may be lightly edited to improve clarity or fit our standards. Each community skill retains its original author credit where applicable.

---

## How to use a skill

1. Open Claude (claude.ai or your Claude desktop app)
2. Navigate to **Skills** in your project or settings
3. Upload or reference the relevant `.md` skill file
4. Claude will apply the skill's instructions automatically during your conversation

## Contributing

We welcome skill submissions from the community. To propose a skill:

1. Fork this repository
2. Add your skill file to the `/skills/community/` directory
3. Open a pull request with a brief description of what the skill does

All submissions are reviewed before merging. We may decline or request changes if a skill doesn't meet our quality bar.
